# SVG Playground

An interactive online SVG editor and playground for creating, editing, and previewing SVG graphics in real-time.

## Features

- **Live Preview**: See your SVG code rendered in real-time as you type
- **Split View**: Work with code and preview side by side
- **Code Formatting**: Auto-indent and format your SVG code
- **Copy & Download**: Easily copy your SVG code or download it as a file
- **Syntax Validation**: Get instant feedback on invalid SVG syntax

## Usage

1. Visit the [SVG Playground](https://rifaterdemsahin.github.io/svg/)
2. Paste or write your SVG code in the editor
3. See the live preview on the right
4. Use the toolbar to format, copy, or download your SVG
